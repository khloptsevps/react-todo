# Список задач

### Приложение для отслеживания списка задач.

Стек:

- React
- Redux-toolkit
- Firebase

---

## Системные требования / System requirements

```sh
node 18.x
firebase
```

## Установка / Setup

```sh
$ make install
```

## Конфигурация / Config

```sh
Нужно настроить подключение к своему приложению в файле firebaseConfig.js.
```

## Запуск (режим разработки) / start (developer mode)

```sh
$ make start
```

## Сборка / Build

```sh
$ make build
```
